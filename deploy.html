🚀 部署指南

将马克思主义答题挑战游戏部署到线上

✅ 项目已准备就绪！ 所有必要的配置文件已创建，您可以选择以下任一方式进行部署。

🌟 方案一：Vercel部署（推荐）

步骤1： 安装Node.js
访问 https://nodejs.org/ 下载并安装最新版本
步骤2： 安装Vercel CLI
npm install -g vercel
步骤3： 部署项目
vercel --prod
步骤4： 按照提示完成部署，获得在线链接
访问Vercel官网

📱 方案二：Netlify部署（最简单）

步骤1： 访问 netlify.com
步骤2： 注册账号并登录
步骤3： 直接拖拽整个项目文件夹到Netlify页面
步骤4： 等待自动部署完成，获得在线链接
访问Netlify

🐙 方案三：GitHub Pages部署

步骤1： 将项目上传到GitHub仓库
步骤2： 在仓库设置中找到"Pages"选项
步骤3： 选择主分支作为源
步骤4： 访问生成的链接：https://你的用户名.github.io/仓库名/code.html
访问GitHub

💻 本地预览

使用Python：
python -m http.server 8080
然后访问：http://localhost:8080/code.html
⚠️ 注意事项：
  • 确保所有文件都在同一个文件夹中
  • 部署后主页面是 code.html
  • 游戏数据会自动保存在浏览器本地存储中
  • 建议使用HTTPS协议访问以获得最佳体验
