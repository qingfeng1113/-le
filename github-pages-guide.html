🐙 GitHub Pages 部署指南

将马克思主义答题挑战游戏部署到GitHub Pages

📋 准备工作：
  • 需要一个GitHub账号（免费注册）
  • 项目文件已准备完毕
  • 了解基本的Git操作（可选，也可以使用网页界面）

1创建GitHub仓库

1.1 访问 GitHub.com 并登录
1.2 点击右上角的 "+" 按钮，选择 "New repository"
1.3 填写仓库信息：
  • Repository name: marxism-quiz-game（或其他名称）
  • Description: 马克思主义答题挑战游戏
  • 选择 Public（公开仓库，免费用户必须选择）
  • 勾选 "Add a README file"
1.4 点击 "Create repository" 创建仓库

2上传项目文件

⚠️ 重要： 确保以下文件都在项目文件夹中：
  • code.html（主游戏文件）
  • effects_demo.html（特效演示）
  • README.md（项目说明）
  • 其他配置文件

方法一：使用GitHub网页界面（推荐新手）

2.1 在仓库页面点击 "uploading an existing file"
2.2 拖拽所有项目文件到上传区域
2.3 在底部填写提交信息："Initial commit - 上传游戏文件"
2.4 点击 "Commit changes" 提交文件

方法二：使用Git命令行

git clone https://github.com/你的用户名/仓库名.git cd 仓库名 # 将所有项目文件复制到这个文件夹 git add . git commit -m "Initial commit - 上传游戏文件" git push origin main

3启用GitHub Pages

3.1 在仓库页面点击 "Settings" 选项卡
3.2 在左侧菜单中找到并点击 "Pages"
3.3 在 "Source" 部分选择：
  • Source: Deploy from a branch
  • Branch: main（或master）
  • Folder: / (root)
3.4 点击 "Save" 保存设置

4配置自定义首页

4.1 由于我们的主文件是 code.html，需要创建一个 index.html 文件作为首页
4.2 在仓库中创建新文件 "index.html"，内容如下：
<!DOCTYPE html> <html lang="zh-CN"> <head> <meta charset="UTF-8"> <meta http-equiv="refresh" content="0; url=code.html"> <title>马克思主义答题挑战</title> </head> <body> <p>正在跳转到游戏页面...</p> <p><a href="code.html">如果没有自动跳转，请点击这里</a></p> </body> </html>

5访问您的网站

5.1 等待几分钟让GitHub Pages构建完成
5.2 访问您的网站地址：
https://你的用户名.github.io/仓库名/
5.3 直接访问游戏页面：
https://你的用户名.github.io/仓库名/code.html
🎉 部署成功！ 现在您可以将链接分享给同学们了。
📝 注意事项：
  • GitHub Pages 可能需要几分钟时间来部署
  • 每次更新文件后，网站会自动重新部署
  • 免费版GitHub Pages有一些使用限制（通常足够个人使用）
  • 确保仓库是公开的（Private仓库需要付费版才能使用Pages）
🔧 常见问题解决：
  • 404错误： 检查文件名是否正确，确保 code.html 存在
  • 样式丢失： 确保所有CSS都在HTML文件内部
  • 更新不生效： 清除浏览器缓存或等待几分钟
